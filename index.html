🏨 여행 예약 앱

Travel Booking App - Flutter Web Application

✅ 앱이 성공적으로 빌드되었습니다!

Flutter 웹 앱이 정상적으로 컴파일되어 정적 파일로 생성되었습니다.

🌐 접속 방법

다음 중 하나의 방법으로 앱에 접속할 수 있습니다:

  1. Flutter 개발 서버: flutter run -d chrome --web-port=8080
  2. 정적 파일 서버: build/web 폴더의 파일들을 웹 서버에 배포
  3. 로컬 서버: Python, Node.js 등의 웹 서버 사용

🏠 메인 화면

  • 상단 검색바와 알림 아이콘
  • 카테고리 섹션 (숙소, 항공, 투어 등)
  • 메인 배너 (일본 여행 특가)
  • 웰컴 기프트 섹션
  • 초특가 바로가기 배너

🎯 초특가 바로가기

  • 일본 로컬 (파란색) → 이벤트 혜택
  • 최 주말한정 (청록색) → 이벤트 혜택
  • 전 세계 (주황색) → 이벤트 공지사항

🏨 호텔 상세 페이지

  • 호텔 정보 (등급, 평점, 편의시설)
  • 결제 혜택 (카드별 할인 정보)
  • 숙소 위치 (지도, 주소)
  • 객실 선택 (3개 객실 옵션)
  • 근처 숙소 둘러보기

📢 이벤트 공지사항

  • 상단 특별 이벤트 배너
  • 카테고리 필터 (전체, 이벤트, 공지 등)
  • 공지사항 리스트 (NEW 배지)
  • 검색 기능 및 상세보기

🎁 이벤트 혜택

  • 상단 특별 혜택 배너
  • 진행중/종료 탭 구분
  • 혜택 카드 (할인율, 기간, 상태)
  • 필터 옵션 및 혜택 적용

📱 반응형 디자인

  • 모바일과 데스크톱 지원
  • 터치 친화적 인터페이스
  • 명확한 시각적 피드백
  • 직관적인 네비게이션

🔧 기술 스택

Flutter + Dart + Material Design
